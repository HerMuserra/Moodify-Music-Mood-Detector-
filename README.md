- Cute, smart, and music-vibey  

## How to Run

1. **Clone the repo:**
bash
git clone <repo-url>
cd Moodify

2.**Install dependencies:**
pip install -r requirements.txt


3.**Run the predictor:**
python mood_predictor.py

4.Type in lyrics and see the mood instantly!
Type exit or quit to stop the program
